These are the data sets containing data of the four Data Types to see if there were differences in
the number of participants and observations across Data Type options, which was the case: 

- `df_CA.RDS` Cross-sectional all waves: 138,807 participants across 336,804 observations.
- `df_CW1.RDS`Cross-sectional Wave 1: 30,416 participants across 30,416 observations.
- `df_LNS.RDS` Longitudinal no strict baseline: 87,029 participants across 285,026 observations.
- `df_LW1.RDS` Longitudinal Wave 1: 23,301 participants across 91,935 observations.
